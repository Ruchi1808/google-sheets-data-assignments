# Google Sheets Data Assignments

This repository contains hands-on data analysis assignments completed using
Google Sheets. The work demonstrates practical skills in handling raw data,
performing lookups, and summarizing information using Pivot Tables.

## Assignments Included

### 1. Marks & Percentage Analysis
- Calculated total marks and percentages
- Scaled scores to 100
- Assigned grades based on criteria

### 2. Data Fetching using VLOOKUP
- Retrieved missing values from a reference sheet
- Used VLOOKUP with proper cell locking
- Ensured data accuracy and consistency

### 3. Data Summarization using Pivot Tables
- Converted raw student data into structured reports
- Grouped data by batch, name, and registration number
- Calculated average and cumulative scores
